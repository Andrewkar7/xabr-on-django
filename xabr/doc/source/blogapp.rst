Blogapp module documentation
=================================================
Приложение личного кабинета пользователя, в котором отображена структура личного кабинета и возможности создавать, просматривать,
редактировать, удалять свои статьи пользователем сайта. Поддерживает вывод статей
с текущим статусом (опубликована, на модерации, черновик).
Поддерживает редактирование информации о пользователе.

class BlogListView(ListView):
    """класс для вывода списка всех статей пользователя"""


class BlogDetailView(DetailView):
    """класс для вывода черновиков пользователя"""


class BlogCreateView(CreateView):
    """класс для вывода страницы создания новой статьи пользователем"""


class BlogUpdateView(UpdateView):
    """класс для вывода страницы редактирования статьи/черновика пользователем"""


class BlogDeleteView(DeleteView):
    """класс для вывода страницы удаления статьи/черновика пользователем"""
