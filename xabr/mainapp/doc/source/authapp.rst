Authapp module documentation
=================================================
Приложение, содержащее информацию о пользователях сайта, в котором отображена структура поддерживаемых
возможностей пользователей - регистрация, авторизация, полномочия пользователя.


Запускаемый модуль,содержит классы переопределения модели пользователя, отображающихся в административной панели
